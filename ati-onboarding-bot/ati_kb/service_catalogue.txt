=== MOBILE APP DEVELOPMENT ===
ATI builds native and cross-platform mobile applications for iOS and Android.

Platforms and approaches:
- Native iOS (Swift) and Android (Kotlin/Java)
- Cross-platform development (React Native, Flutter)
- Progressive Web Apps (PWA) for mobile-friendly experiences

Common mobile app features ATI delivers:
- User authentication and secure login (biometrics, SSO)
- Push notifications and in-app messaging
- Offline mode and local data sync
- Payment processing and in-app purchases
- GPS, maps, and location-based services
- Camera, photo upload, and document scanning
- API integration with backend systems and third-party services
- Admin dashboards and analytics

Mobile app project workflow:
Discovery → UX/UI design → Prototyping → Development → QA testing → App Store / Play Store submission → Ongoing maintenance and updates

Typical onboarding questions for mobile projects:
- Target platforms (iOS, Android, or both)
- Target users and primary use cases
- Must-have features for version 1 (MVP)
- Existing backend or API to connect to
- Timeline and launch date
- Design references or brand guidelines
